About this site

This site was made with Flask and Frozen-Flask in Python!

I employed Flask because it makes building web pages straightfoward. Frozen-Flask automatically converts the project into static HTML, which can then be published into GitHub. The details can be found in both branches of this site's repository, where a custom GitHub action compiles the main branch into gh-pages for deployment.
